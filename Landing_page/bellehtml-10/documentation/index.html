CSS Files

  • bootstrap.min.css - bootstrap framework styles
  • font-awesome.min.css - styles for icon font
  • animate.css - tool for animation
  • style.css - main theme styles
  • responsive.css - style for all screens




All CSS files located in css directory.

You can setup new css file globally by importning it to the top of the style.css file. Just add line:
